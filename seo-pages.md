# SEO Pages Reference

Target keywords, metadata, and schema requirements per page.
Read this when working on SEO metadata, title tags, or schema markup.

---

## SEO requirements — every page

- Unique `<title>` tag
- Unique `<meta name="description">`
- `og:title`, `og:description`, `og:image`
- Schema.org structured data (type varies by page — see below)
- Descriptive `alt` text on all images
- Canonical tag where needed

## Schema — key structured data

### LocalBusiness (site-wide / homepage)
```json
{
  "@context": "https://schema.org",
  "@type": "TouristAttraction",
  "name": "Manchester River Cruises",
  "description": "Passenger boat tours on the Manchester Ship Canal and River Irwell",
  "url": "https://www.manchesterrivercruises.com",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "Millennium Footbridge, Salford Quays",
    "postalCode": "M50 3RB",
    "addressCountry": "GB"
  },
  "geo": {
    "@type": "GeoCoordinates",
    "latitude": 53.4726,
    "longitude": -2.2979
  }
}
```

### FAQPage `/faq`
Use `@type: FAQPage` with `mainEntity` array of `Question` items.

### Article (guide pages)
Use `@type: Article` with `author`, `datePublished`, `headline`.
